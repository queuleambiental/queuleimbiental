# Queule Ambiental

Marketing website for **Queule Ambiental**, an environmental waste management company based in Santiago, Chile. The site presents their services, pricing model, coverage areas, and contact information.

## Tech Stack

- **Framework:** TanStack Start (React 19 + TanStack Router v1)
- **Build:** Vite 7
- **Styling:** Tailwind CSS 4 + custom CSS properties
- **Language:** TypeScript 5.7 (strict mode)
- **Deployment:** Netlify

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts at `http://localhost:3000` (or `http://localhost:8888` via Netlify CLI).

### With Netlify CLI (recommended)

```bash
netlify dev
```
